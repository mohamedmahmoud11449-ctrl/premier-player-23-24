# Premier League 2023/24 Data Analysis Project

This repository contains a complete data analysis project focused on Premier League 2023/24 player and team performance.
The goal of the project is to transform raw football statistics into clear, actionable insights using data analytics and visualization techniques.

## Project Objectives
- Analyze team and player performance during the 2023/24 Premier League season
- Compare actual vs expected metrics to evaluate efficiency
- Identify key performance indicators (KPIs) for teams and players
- Build interactive dashboards to support data-driven insights

## Key Analyses
- Team Attacking Performance (Goals, xG, xAG, per 90)
- Player Performance (Goals, Assists, expected metrics)
- Actual vs Expected Output
- Ball Progression (Progressive passes, runs, carries)
- Player Usage (Minutes played)

## Tools & Technologies
- SQL: data exploration, cleaning, and validation
- Power BI: data modeling, DAX measures, and dashboard creation

## Skills Demonstrated
- Sports data analysis
- KPI design and metric validation
- Data cleaning and quality checks
- Data storytelling and dashboard structuring
